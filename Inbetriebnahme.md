# Inbetriebnahme:
An Strom anschließen, nach etwa 1 Minute wird für 15 Minuten ein Accesspoint mit dem Namen „ABBRANDREGELUNG_XXXXXX“ erstellt. Das Passwort ist der Name des W-LAN-Netzes.
Nach Verbindung mit dem Netzwerk öffnet sich automatisch eine Konfigurationsseite. Falls nicht (Windows 10) die IP 192.168.4.1 im Browser eingeben. Unter „Configuration“ das passende Netzwerk auswählen (Button „Configuration“ nur einmal drücken und 5-10s warten).

Webserver Zugangsdaten:
Werkseinstellung: Benutzer= „User“, Passwort = „Passwort“
Der Benutzername muss zwischen 3 und 32 Zeichen, das Passwort zwischen 3 und 64 Zeichen lang sein. Unzulässige Zeichen sind:  " \ < > , ;

Zurücksetzen auf Werkseinstellung: 
-Vom Strom trennen, Reset-Knopf drücken und halten, an Strom anschließen. Reset-Knopf so lange halten, bis die LED Rot blinkt. Zugangsdaten zum Webserver, W-Lan-Daten und Servo-Einstellungen werden gelöscht.

LED-Anzeige:
|Farbe	|Bedeutung
|---------------|--------
|Grün|	Alles OK
|Rot|	Keine WLAN/Internet-Verbindung
|Gelb|	CO-Sensor defekt/nicht angeschlossen
|Blau|	Sensor Abgastemperatur defekt/nicht angeschlossen
|Rot Blinkend|	Hardwarefehler
